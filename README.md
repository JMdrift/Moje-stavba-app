# MojeStavba_v0.11.5_017_DashboardNoScrollPlusHUD

Základ: předchozí Blueprint HUD dashboard.

## Úpravy
- Dashboard upraven tak, aby se vešel na jednu obrazovku bez běžného scrollování.
- Zachován layout a logika, zmenšeny pouze výšky, mezery a proporce prvků.
- Den stavby zůstává výrazný HUD prvek, bez bubliny.
- Rychlé centrální plus převedeno do technického blueprint/HUD stylu.
- Plus menu už nepůsobí jako kulatý koláč, ale jako ostrý stavební panel.
- Logika rychlého plus zůstává stejná: Zápis / Fotky / Transakce / Událost.
